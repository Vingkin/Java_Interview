# 基本操作

## 移动光标的方法

| 命令     | 操作                                         |
| -------- | -------------------------------------------- |
| +        | 光标移动到非空格符的下一行                   |
| -        | 光标移动到非空格符的上一行                   |
| n<space> | 10<space>光标移动到当前行的第10个字符        |
| **0**    | **光标移动到当前行最前面**                   |
| **$**    | **光标移动到当前行最后面**                   |
| H        | 光标移动到这个屏幕的最上方那一行的第一个字符 |
| M        | 光标移动到这个屏幕的中间一行的第一个字符     |
| L        | 光标移动到这个屏幕的最下方那一行的第一个字符 |
| **G**    | **光标移动到文档最后一行**                   |
| nG       | 光标移动到第n行                              |
| **gg**   | **光标移动到文档第一行**                     |
| n<Enter> | 光标向下移动n行                              |

## 复制、粘贴、删除

| 命令      | 操作                                                         |
| --------- | ------------------------------------------------------------ |
| y         | 复制                                                         |
| p         | 粘贴                                                         |
| x, X      | 在一行字当中，x 为向后删除一个字符 (相当于 [del] 按键)， X 为向前删除一个字符(相当于 [backspace] 亦即是退格键) |
| nx        | 向后连续删除n个字符                                          |
| dd        | 删除当前行                                                   |
| ndd       | 向下删除n行                                                  |
| dgg / d1G | 删除光标所在到第一行的所有数据                               |
| dG        | 删除光标所在到最后一行的所有数据                             |
| d$        | 删除游标所在处，到该行的最后一个字符                         |
| d0        | 那个是数字的 0 ，删除游标所在处，到该行的最前面一个字符      |
| yy        | 复制游标所在的那一行                                         |
| p, P      | p 为将已复制的数据在光标下一行贴上，P 则为贴在游标上一行     |
| J         | 将光标所在行与下一行的数据结合成同一行                       |
| u         | 撤销                                                         |
| ctrl+r    | redo                                                         |
| .         | 重复之前的删除或者粘贴等操作                                 |

## 翻页

| 命令   | 操作     |
| ------ | -------- |
| ctrl+d | 下翻半页 |
| ctrl+u | 上翻半页 |
| ctrl+f | 下翻一页 |
| ctrl+b | 上翻一页 |

## 查找与替换

| 命令                      | 操作     | 备注                           |
| ------------------------- | -------- | ------------------------------ |
| /{pattern}                | 查找     | 使用n跳转下一个，N跳转上一个   |
| :%s/{parttern}/{string}/g | 全局替换 | 将g替换成c则会一个一个进行确认 |



# 进阶操作

# ## 基于单词的移动

| 命令 | 光标动作                          |
| ---- | --------------------------------- |
| w    | 正向移动到下一个单词开头          |
| b    | 反向移动到当前单词/上一单词的开头 |
| e    | 正向移动到当前单词/上一单词的结尾 |
| ge   | 反向移动到上一单词的结尾          |

## 对字符进行查找通过查找进行移动

> 这6个操作只是针对当前行

| 命令    | 光标动作                                   |
| ------- | ------------------------------------------ |
| f{char} | 正向移动到下一个{char}所在之处             |
| F{char} | 反向移动到上一个{char}所在之处             |
| t{char} | 正向移动到下一个{char}所在之处的前一个字符 |
| T{char} | 反向移动到上一个{char}所在之处的后一个字符 |
| ;       | 重复上次的字符查找命令                     |
| ,       | 反向重复上次的字符查找命令                 |

## 分隔符文本对象 / 范围文本对象（重点）

> 下面命令都是在视图模式中执行的 `v`
>
> 只用小括号来举例，其他的都一样
>
> i--in a--around

| 命令 | 选择区域                             |
| ---- | ------------------------------------ |
| a)   | 一对圆括号及其内部                   |
| i)   | 圆括号内部                           |
| at   | t表示xml标签<xml>tag</xml>，包含标签 |
| it   | 标签内部                             |

| 命令 | 选择范围           |
| ---- | ------------------ |
| iw   | 当前单词           |
| aw   | 当前单词及一个空格 |
| iW   | 当前字串           |
| aW   | 当前字串及一个空格 |
| is   | 当前句子           |
| as   | 当前句子及一个空格 |
| ip   | 当前段落           |
| ap   | 当前段落及一个空行 |

## 操作符待决模式

> {motion}指的就是：分隔符文本对象和范围文本对象

| 命令      | 操作     | 备注                       |
| --------- | -------- | -------------------------- |
| d{motion} | 删除模式 | dd删除一行 diw删除当前单词 |
| c{motion} | 修改模式 | cc修改一行 ciw修改当前单词 |
| y{motion} | 复制模式 | yy复制一行 yiw复制当前单词 |
| v{motion} | 可视模式 |                            |

| 命令    | 操作     |
| ------- | -------- |
| m{mark} | 设置标记 |
| `{mark} | 返回标记 |
